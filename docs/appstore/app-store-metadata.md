### 説明(4000文字以内)

```
ichiseEdit は、iPad のためのシンプルで軽快なテキストエディタです。余計な機能を削ぎ落とし、「開いて、書いて、勝手に保存される」という基本の体験を大切に作りました。

◆ 安心して使える
・完全無料。広告なし、アプリ内課金なし、アカウント登録不要
・データ収集なし。文書があなたの選んだ保存先の外に送られることはありません

◆ 「ファイル」アプリと完全連携
・iCloud Drive、iPad 本体、対応クラウドの .txt ファイルをそのまま開いて編集
・自動保存に対応。保存ボタンを押す必要はありません
・複数ウィンドウ対応。Split View や Stage Manager で 2 つの文書を並べて編集できます

◆ 編集機能
・検索・置換(iPadOS 標準の検索バー。「すべて置換」にも対応)
・外部キーボードのショートカットに対応(⌘F 検索、⌥⌘F 置換、⌘+ / ⌘− 文字サイズ変更 など)
・フォントサイズ調整(設定は自動的に保存)
・文字数・行数を常時表示
・ダークモード対応

◆ 軽快な動作
・1MB 級の大きなテキストファイルでも、スクロールや入力がもたつかないよう設計し、検証しています
・日本語入力で快適に使えるよう調整しています

ちょっとしたメモから長文の下書きまで、iPad でのテキスト編集にお使いください。
```
