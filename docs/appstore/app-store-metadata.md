### 説明(4000文字以内)

```
ichiseEdit は、iPad のためのシンプルで軽快なテキストエディタです。「開いて、書いて、勝手に保存される」という基本の体験を大切にしつつ、必要になったら Markdown・コード編集・自作マクロまで――使うほど応えてくれるように作りました。

◆ 安心して使える
・完全無料。広告なし、アプリ内課金なし、アカウント登録不要
・データ収集なし。文書があなたの選んだ保存先の外に送られることはありません
・マクロもすべて端末内・サンドボックス内で動作。ネットワーク送信はありません

◆ 「ファイル」アプリと完全連携
・iCloud Drive、iPad 本体、対応クラウドのファイルをそのまま開いて編集
・自動保存に対応。保存ボタンを押す必要はありません
・複数ウィンドウ対応。Split View や Stage Manager で 2 つの文書を並べて編集できます

◆ 基本の編集機能
・検索・置換(iPadOS 標準の検索バー。「すべて置換」にも対応)
・外部キーボードのショートカットに対応(⌘F 検索、⌥⌘F 置換、⌘+ / ⌘− 文字サイズ変更 など)
・フォントサイズ調整、行番号表示、ワードラップ(折り返し)の ON/OFF
・文字数・行数を常時表示
・ダークモード対応

◆ Markdown
・見出し・強調・コード・リンクなどをリアルタイムに色分け表示
・太字・見出し・リンクなどをワンタップで挿入する編集コマンド
・書きながらプレビューで確認

◆ コード編集
・Swift / Python / JavaScript / Ruby / Go / Rust / Kotlin / C 系ほか、
　Common Lisp・Scheme・Clojure・ISLISP・Emacs Lisp・Haskell・LaTeX を含む多言語のシンタックスハイライト
・行番号表示と自動インデント

◆ 長文執筆支援
・文字数・単語数・段落数などの詳細な統計表示
・集中モードで今書いている段落以外を淡く表示
・選択範囲の文字数表示

◆ マクロで自分好みに拡張(ISLISP)
・エディタ内蔵の ISLISP インタプリタで、自分だけのコマンドを作れます
・Lisp マクロとオブジェクト指向(ILOS)に対応した本格的な処理系
・テキスト操作・検索・ファイル操作・iPadOS 連携(読み上げ・共有・辞書など)のエディタ API
・作ったコマンドはメニューに追加でき、キーボードショートカットも割り当て可能
・その場で試せる REPL コンソール
・正規表現に対応した grep(このファイル / フォルダ再帰)
・書き方はアプリ内マニュアルに収録。標準コマンドやサンプルマクロも同梱

◆ 軽快な動作
・1MB 級の大きなテキストファイルでも、スクロールや入力がもたつかないよう設計し、検証しています
・日本語入力で快適に使えるよう調整しています

ちょっとしたメモから、Markdown 文書、コードの下書き、長文の執筆、さらには自分専用ツールづくりまで。iPad でのテキスト編集に、ずっと使えるエディタです。
```

### Description (max 4000 chars)

```
ichiseEdit is a simple, fast text editor built for iPad. It keeps the essentials front and center — open, write, and let your work save itself — while growing with you: Markdown, code editing, and your own macros are there when you need them.

WORRY-FREE
• Completely free. No ads, no in-app purchases, no account required
• No data collection. Your documents never leave the storage location you choose
• Macros run entirely on-device, inside a sandbox — nothing is sent over the network

WORKS WITH THE FILES APP
• Open and edit files directly from iCloud Drive, on-device storage, and supported cloud providers
• Autosave keeps your work safe — there is no save button to remember
• Multiple windows: edit two documents side by side in Split View or Stage Manager

CORE EDITING
• Find and replace with the standard iPadOS find bar, including Replace All
• Hardware keyboard shortcuts (⌘F find, ⌥⌘F replace, ⌘+ / ⌘− text size, and more)
• Adjustable font size, line numbers, and word wrap on/off
• Always-visible character and line count
• Dark Mode support

MARKDOWN
• Real-time highlighting of headings, emphasis, code, links, and more
• One-tap editing commands for bold, headings, links, and more
• Preview as you write

CODE EDITING
• Syntax highlighting for many languages — Swift, Python, JavaScript, Ruby, Go, Rust, Kotlin, C-family, plus Common Lisp, Scheme, Clojure, ISLISP, Emacs Lisp, Haskell, and LaTeX
• Line numbers and automatic indentation

LONG-FORM WRITING
• Detailed statistics: characters, words, paragraphs, and more
• Focus mode dims everything but the paragraph you are writing
• Character count for the current selection

EXTEND IT YOURSELF (ISLISP MACROS)
• A built-in ISLISP interpreter lets you create your own commands
• A real language with Lisp macros and an object system (ILOS)
• An editor API for text, search, files, and iPadOS features (speech, share, dictionary, and more)
• Add your commands to the menu and assign keyboard shortcuts
• A REPL console to try things instantly
• grep with regular expressions (this file / recurse a folder)
• An in-app manual, standard commands, and sample macros included

FAST AND RESPONSIVE
• Designed and tested to stay smooth while scrolling and typing, even in large 1 MB text files

From quick notes to Markdown documents, code drafts, long-form writing, and even building your own tools — ichiseEdit is an editor you can keep using on iPad.
```
